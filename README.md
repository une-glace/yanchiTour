# 盐池旅游导览示例网站

本项目是根据“盐池旅游导览”小程序思路改写的一个静态网页示例，主要用于**宣传展示**和**界面结构演示**。  
页面结构参考了小程序的设计思路：顶部为滚动风景展示区域，下方为功能入口卡片，分别跳转到景点、美食、住宿、交通和行程等页面。

> 说明：当前所有风景图片均使用在线随机占位图服务生成，仅用于占位与结构展示，后续可替换为真实图片与正式文案。

## 技术栈

- 纯静态网页：HTML + CSS
- 原生 JavaScript（无框架依赖）
- 不需要构建工具或后端服务，适合作为静态宣传页部署

## 目录与主要文件

- `index.html`：网站首页  
  - 顶部盐池风光轮播区域（自动切换 + 底部小圆点）  
  - 五个导航入口：景点 / 美食 / 住宿 / 交通 / 行程  
  - 网站介绍说明
- `attractions.html`：景点列表页  
  - 卡片列表展示若干盐池及周边示例景点  
  - 卡片包含占位图片、景点名称、位置、标签和简介  
  - 点击卡片进入详情页
- `attraction-detail.html`：景点详情页  
  - 顶部多图轮播展示当前景点图片（占位图）  
  - 下方分块展示：概览 / 历史与人文 / 地理位置与环境 / 特色体验与特产
- `food.html`：美食页面占位，用于承接首页“美食”入口
- `stay.html`：住宿页面占位，用于承接首页“住宿”入口
- `transport.html`：交通页面占位，用于承接首页“交通”入口
- `plan.html`：行程规划页面占位，用于承接首页“行程”入口
- `style.css`：全站样式文件（首页、列表页、详情页等通用）

## 本地预览方式

本项目是静态站点，直接在浏览器中打开首页文件即可预览。

**方式一：双击打开**

1. 在资源管理器中进入项目目录：`e:\sync\26冬宁夏清华实践\小程序\`
2. 双击 `index.html`  
3. 浏览器会自动打开首页（地址栏一般为 `file:///...`）

**方式二：浏览器中打开文件**

1. 打开浏览器（例如 Edge / Chrome）  
2. 使用快捷键 `Ctrl + O`  
3. 选择 `index.html` 并打开
